# Config Layout

## Active

Use only the files in `configs/active/` for normal work:

- `release_eval.yaml`
- `train_mains5_all10.yaml`
- `house_overrides_daily.example.yaml`
- `house_overrides_daily_denoise.yaml`

## Archive

`configs/archive/` contains historical experiment configs kept only for reference.

They are not the recommended entrypoints for new runs or deployment.
